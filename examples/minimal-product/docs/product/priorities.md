# Product Priorities

## Must Win
- visitor can submit a request in under 2 minutes
- owner can review requests in one queue
- each request has enough context to accept, reject, or follow up

## Important but Later
- calendar sync
- automated reminders
- analytics dashboard

## Explicitly Deferred
- payment capture
- team collaboration
- public marketplace discovery

## Trade-off Notes
Optimize for operator clarity and request quality over broad surface area.
