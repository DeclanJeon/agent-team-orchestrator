# Social Preview Setup

GitHub social preview images cannot be set purely by committing a file into the repository.
You still need to upload the image once in the repository settings.

Recommended asset
- `assets/social-preview.svg`

Manual setup steps
1. Open repository settings:
   `https://github.com/DeclanJeon/agent-team-orchestrator/settings`
2. Scroll to the Social preview section.
3. Upload `assets/social-preview.svg`.

What the image is designed to communicate
- Hermes skill pack
- document-first multi-agent product organization
- role boundaries, handoff contracts, review gates, and one-click bootstrap

If you later change the repository positioning, update both:
- `assets/social-preview.svg`
- README tagline / repo description
